## 页面结构

### 1. 今日视图（Dashboard）
- 今日任务摘要
- 进行中的番茄钟
- 快速添加任务入口
- 日历缩略图

### 2. 任务管理
- 列表视图：按优先级/标签/截止日分组
- 看板视图：按状态（待办/进行中/已完成）拖拽
- 任务详情侧边栏：标题、描述、子任务、标签、提醒

### 3. 日历视图
- 日/周/月视图切换
- 拖拽创建/调整事件
- 任务与学习计划叠加显示

### 4. 学习计划
- 目标看板：学习目标卡片
- 进度追踪：完成率、学习时长统计
- 计划编辑器：时间段、科目、频率设置

### 5. 番茄钟
- 计时器界面（大数字 + 进度环）
- 任务关联选择
- 历史记录列表

### 6. 备忘录
- 笔记列表 + 编辑器
- 支持 Markdown
- 可关联任务/学习计划

### 7. 全局搜索（Command Palette）
- 快捷键触发（Ctrl+K / Cmd+K）
- 模糊搜索任务/备忘/学习计划

### 8. 设置
- 主题切换（深色/浅色）
- 快捷键配置
- 数据导入/导出
- 关于

## 交互流程

```
用户启动 → 首页（今日视图）
   │
   ├─→ 快速添加任务 → 任务列表更新
   │
   ├─→ 点击日历 → 日历视图 → 拖拽安排
   │
   ├─→ 切换到学习 → 学习计划管理
   │
   └─→ 开始番茄钟 → 关联任务 → 计时
```

## 组件复用

- TaskCard：任务卡片组件
- CalendarEvent：日历事件组件
- ProgressRing：进度环组件
- CommandPalette：命令面板组件
- SidePanel：侧边详情面板
